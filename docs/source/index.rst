Welcome to SPURS's documentation!
================================

SPURS is a accurate, rapid, scalable, and generalizable stability predictor. This repository is an official implementation of the paper `Rewiring protein sequence and structure generative models to enhance protein stability prediction <https://www.biorxiv.org/content/10.1101/2025.02.13.638154v1>`_.
